# lake
A build tool styled after make but written and scriptable in Lua. Rejoice!

[![Build Status](https://travis-ci.org/lua-lake/lake.svg?branch=master)](https://travis-ci.org/lua-lake/lake)

## Setup
https://luvit.io/install.html

## Run sample
```shell
cd sample
luvi .. -- sample.lk
```

or

```shell
lit make
cd sample
../lake sample.lk
```
